JaxMe 2 is an open source implementation of JAXB, the specification for Java/XML binding.

JaxMe contains the following components:

  • JaxMeAPI is a clean room implementation of the JAXB API
  • JaxMeJS is the JaxMe java source generation framework
  • JaxMeXS is the JaxMe parser for XML schema.
  • JaxMePM is the JaxMe persistence management framework
